## 🛠️ Tech Stack Checklist for Post Tags
- **Backend:** Java 8+, Jakarta EE (Servlets 5.0)
- **Frontend:** JSP, Glassmorphism CSS, HTML5, Remix Icon Library
- **Database:** MySQL (JDBC & DAO Pattern)
- **Analytics:** Chart.js
- **Build/Cloud:** Maven, Apache Tomcat 10.0+
- **Integrations:** Email API for status notifications (simulated/logic)

## 🔄 Core App Flow (The "Happy Path")
1.  **Onboarding:** Recruiters register and wait for **Admin approval**. Students create profiles and upload PDF resumes.
2.  **Job Posting:** Approved Recruiters post job drives with specific **CGPA eligibility** and deadlines.
3.  **Application:** Students browse jobs; the system dynamically shows eligibility based on their stored CGPA. They apply with one click.
4.  **Selection:** Recruiters review applicants and resumes. Updating a status to "Selected" triggers an **automated email notification** to the student.
5.  **Analytics:** Admins monitor the entire funnel through a real-time analytics dashboard powered by Chart.js.

## 🎨 UI/Design Highlights
*   **Glassmorphism:** Frosted-glass effect using `backdrop-filter: blur(12px)` and `rgba(255, 255, 255, 0.85)`.
*   **Animated Backgrounds:** Dynamic backgrounds with `linear-gradient` and floating blurred circles with `float` animations.
*   **Interactive States:** Buttons and inputs with scale-up and shadow depth transitions on hover.
*   **Iconography:** High-quality Remix Icon library for a premium look.
